[b]Solve Topic[/b]
This addon adds a new capability, allowing one to mark a topic solved or unsolved.
[list]
[li]You can select the boards for this feature to be active.[/li]
[li]Permissions include solve own topic and any topic.[/li]
[li]Adds quick moderation option when select boxes are used[/li]
[li]Includes a topic solved log, which can be viewed both from Admin are and Moderation areas.[/li]
[/list]
